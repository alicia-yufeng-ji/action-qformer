# Action QFormer

Project page for **Action QFormer: Structured Representation Shaping under Action Supervision in Vision-Language-Action Models**.

- Project page: https://alicia-yufeng-ji.github.io/action-qformer/
- Paper: https://arxiv.org/abs/2607.14635

## Local preview

Serve the repository root with a local HTTP server, then open the displayed URL in a browser. For example:

```bash
python3 -m http.server 8000
```

The page is implemented in `index.html`, with project styles and behavior in `static/css/index.css` and `static/js/index.js`.

## Assets

- Images: `static/images/`
- Videos: `static/videos/`
- Paper PDF: `static/pdfs/paper.pdf`
